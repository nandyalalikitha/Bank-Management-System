# Bank Management System 💳

## 📌 Project Description
This is a simple Bank Management System developed using Java. It allows users to perform basic banking operations like creating an account, depositing money, withdrawing money, and checking balance.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

## 🛠️ Technologies Used
- Java
- OOP Concepts

## 📂 Project Structure
- Account.java
- Bank.java
- Main.java

## ▶️ How to Run
1. Open project in Eclipse
2. Run Main.java file
